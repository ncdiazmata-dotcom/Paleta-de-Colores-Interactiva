# 🎨 Colorfly Studio

**Colorfly Studio** es un generador cromático avanzado diseñado para la creación de identidades visuales y sistemas de branding de alto impacto. Cuenta con una interfaz moderna de estilo brutalista/industrial inspirada en software de diseño profesional.

## ✨ Características Principales

*   **Generación Inteligente:** Mutación de colores armónicos con un solo clic o presionando la barra espaciadora.
*   **Bloqueo de Muestras:** Fija los colores que te gusten para que no cambien en la siguiente generación.
*   **Edición Manual:** Selector de color integrado en cada tarjeta para ajustar tonalidades con precisión.
*   **Modo Presentación:** Vista limpia a pantalla completa ideal para mostrar paletas a clientes (se sale con la tecla `ESC`).
*   **Persistencia:** Sistema de guardado local e historial de paletas recientes utilizando `LocalStorage`.
*   **Compartición Avanzada:** Genera enlaces dinámicos para compartir tus paletas mediante URLs o a través de la API nativa de tu dispositivo.
*   **Diseño Fluido y Responsivo:** Optimizado para una visualización perfecta en computadoras, tablets y smartphones.

## 🛠️ Tecnologías Utilizadas

*   **HTML5** - Estructura semántica avanzada.
*   **CSS3** - Estilos brutalistas nativos con variables personalizadas y Media Queries para adaptabilidad total.
*   **JavaScript (ES6+)** - Motor cromático basado en manipulación del espacio de color HSL y control del estado de la app sin librerías externas.
*   **FontAwesome** - Set de iconos vectoriales.
